Preconditioners

ExtendableSparse.AbstractExtendablePreconditioner — Type

Abstract type for an extendabe preconditoioner working together with ExtandableSparseMatrix. Still beta, so not in the published documentation.

Any such preconditioner should have the following fields

  extmatrix
  pattern_timestamp

and methods

  update!(precon)

The idea is that, depending if the matrix pattern has changed, different steps are needed to update the preconditioner.

Moreover, they have the ExtendableSparseMatrix as a field, ensuring consistency after construction.
